#  Blood Bank Management System

##  فكرة المشروع
نظام بنك دم رقمي يهدف إلى تنظيم عمليات تخزين وتبرع ونقل الدم بين المتبرعين والمرضى. يساعد في تسجيل بيانات المتبرعين، المرضى، عمليات التبرع، ونقل الدم بدقة مع تتبع الممرضين والمشافي المرتبطة بكل مريض.

---

##  مكونات النظام

###  الممرضين
- الاسم
- العمر
- رقم الهاتف
- تاريخ التوظيف

###  المتبرعين
- الاسم
- العمر
- الجنس
- رقم الهاتف
- زمرة الدم
- تاريخ التبرع
- الممرض المسؤول

###  المرضى
- الاسم
- العمر
- الجنس
- رقم الهاتف
- زمرة الدم
- المشفى المرتبط به
- تاريخ عملية نقل الدم

###  المشافي
- الاسم
- الموقع

###  أنواع الدم المخزنة
- زمرة الدم
- الكمية المتاحة

###  عمليات نقل الدم
- زمرة الدم المطلوبة
- الكمية المنقولة
- تاريخ العملية
- الممرض المسؤول
